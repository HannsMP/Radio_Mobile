# Radio_Mobile

este repo es el resumen de la instalacion oficial de Radio Mobile en español, el cual se encuentra en el siguiente enlace:
https://www.ve2dbe.com/download/download.html

## Informacion de mi maquina
- Windows 11 Pro
- Procesador: AMD Ryzen 5
- RAM 8GB

## Pasos para la instalacion

1. En el explorador de archivos ir a la siguiente ruta: `C:\`

2. Haz `git clone https://github.com/HannsMP/Radio_Mobile`

3. Listo disfruta de la instalacion, el programa se encuentra en la carpeta `C:\Radio_Mobile\src\rmwspa.exe`
